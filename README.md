# User Behavior Analysis Project

## Overview
- This project aims to analyze user behavior, cooking preferences, and order trends based on three datasets: UserDetails, CookingSessions, and OrderDetails.
- The project involves cleaning and merging data, performing exploratory data analysis (EDA), and creating visualizations to derive insights.

## Tasks
- Clean and merge the datasets.
- Perform exploratory data analysis (EDA).
- Identify popular dishes and trends based on user demographics.
- Provide business recommendations to improve user engagement.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Files
- `UserDetails.csv`: Information about the users.
- `CookingSessions.csv`: Data about cooking sessions.
- `OrderDetails.csv`: Information about user orders.
- `Merged_UserData.csv`: Merged dataset.
- `Assignment.ipynb`: Jupyter notebook containing the analysis code.

## Recommendations
- Offer more dinner options based on popularity.
- Target the '30-40' age group with more lunch meals.
